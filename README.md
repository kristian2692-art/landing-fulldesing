# FullDesign - Cinematic Landing Page

Una landing page de alto rendimiento y alta fidelidad diseñada para la marca **FullDesign**. 
Creada con una estética cinematográfica, "pixel perfect", orientada a la conversión y escalamiento de ventas mediante Meta Ads e Inteligencia Artificial.

## Tecnologías Utilizadas
- **React 19**: Construcción de UI por componentes modernos.
- **Vite**: Herramienta de construcción y entorno de desarrollo rápido.
- **Tailwind CSS**: Estilización ágil mediante utilidades de diseño atómico.
- **GSAP 3**: Animaciones avanzadas y "Smooth Scrolling" (ScrollTrigger) en toda la interfaz.
- **Lucide React**: Sistema de iconos limpios y vectorizados.

## Características Principales
- **Diseño Estricto**: Sistema de colores definido (Deep Blue, Electric Cyan, Neon Magenta, Smoke White).
- **Textura Cinematic**: Overlay sutil de "noise" (ruido) orgánico basado en filtros SVG Turbulence.
- **Artefactos de Conversión**: Módulos funcionales interactivos (terminales emuladas, generador simulado por IA).
- **Animaciones fluidas**: Botones magnéticos con respuesta al desplazamiento y entradas de componentes al transitar.

## Instalación y Ejecución Local
Para ejecutar este proyecto de forma iterativa y colaborativa en tu entorno local:

1. **Instalar dependencias**:
```bash
npm install
```

2. **Iniciar el servidor de desarrollo local**:
```bash
npm run dev
```

3. **Construir (Build) los archivos minificados para producción**:
```bash
npm run build
```
